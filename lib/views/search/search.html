WFDrops - Warframe Drop Table Search Tool

WFDrops is a third-party tool used to provide easier access to Warframe's official drop rates. All data is obtained regularly from the official drop tables here. A ratelimited RESTful API is also available, for developer use.

This page is still in active development. Anything here could change at any time. Check the GitHub repo every now and then for more information.
